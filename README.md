# 📰 Fake News Detector using Machine Learning

## 📌 Overview

The Fake News Detector is a Machine Learning-based web application that identifies whether a news article is **Real** or **Fake**.  
The system uses Natural Language Processing (NLP) techniques to analyze text content and classify it based on trained data models.

This project aims to reduce misinformation spread on social media and digital platforms.

---

## 🚀 Features

- 🔍 Detects fake and real news instantly
- 🧠 Uses NLP for text preprocessing
- 📊 Machine Learning-based classification
- 🌐 Simple and user-friendly interface
- 📈 High accuracy prediction model

---

## 🛠️ Tech Stack

- **Frontend:** HTML, CSS
- **Backend:** Python (Flask)
- **Machine Learning:** Scikit-learn
- **Libraries Used:**
  - pandas
  - numpy
  - sklearn
  - nltk
  - pickle

---

## 🧠 Machine Learning Model

The project uses the following steps:

1. Data Collection (Fake & Real News Dataset)
2. Data Cleaning & Preprocessing
3. Feature Extraction using TF-IDF Vectorizer
4. Model Training using:
   - Logistic Regression / Naive Bayes
5. Model Evaluation (Accuracy, Precision, Recall)
6. Deployment using Flask

---
## 📊 Dataset

The dataset contains labeled news articles categorized as:
- Real
- Fake

Common sources:
- Kaggle Fake News Dataset
